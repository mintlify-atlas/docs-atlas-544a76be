# Welcome to Softwart Backend

Softwart Backend is a powerful CLI tool and TypeScript framework that generates complete Express.js backend applications with TypeORM, following the MVC architecture pattern. Generate models, controllers, and routes automatically from schema definitions and focus on business logic instead of boilerplate.

## What's Included

The framework provides a complete backend solution:

- **CLI Scaffolding** - Create new projects with `create-backend-app`
- **Code Generators** - Auto-generate models, controllers, and routes
- **Authentication System** - JWT-based auth with role management
- **CRUD Operations** - Full Create, Read, Update, Delete for all entities
- **Middleware Stack** - CORS, rate limiting, error handling, and validation
- **Database Management** - TypeORM with PostgreSQL, seeding support
- **Type Safety** - Full TypeScript support throughout

## Use Cases

Softwart Backend is ideal for:

- **Rapid Prototyping** - Get a working API up in minutes
- **Business Applications** - Built-in support for complex entity relationships
- **SaaS Backends** - Authentication, authorization, and user management included
- **Learning Projects** - Clean MVC architecture for educational purposes
- **Microservices** - Quick service scaffolding with consistent patterns

## Next Steps

<Steps>
  <Step title="Install the CLI">
    Install Softwart Backend globally via npm
    
    ```bash
    npm install -g backend-softwart-mvc
    ```
  </Step>
  
  <Step title="Create Your First Project">
    Generate a new backend project
    
    ```bash
    create-backend-app my-project
    ```
  </Step>
  
  <Step title="Generate Code">
    Use the generators to create models, controllers, and routes
    
    ```bash
    node generate-model.js
    node generate-controllers.js
    node generate-routes.js
    ```
  </Step>
</Steps>
